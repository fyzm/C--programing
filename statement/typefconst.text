const int n = 10;
const int tax_brackets[] = {750,2250,3750}
//
1.const 是文档格式:声明对象是const类型可以提示任何阅读程序的人,该对象值不会变
2.编译器可以检查到试图改变对象的值
3.当为特定类型的应用编写程序时，编译器可以用const来识别需要存储到ROM
4.可以用#define指令为数字

声明符
1.使用*开头的声明符表示指针
2.使用[]结尾的声明符表示数组
3.使用()声明表示的是函数
解释复杂声明
始终从内往外读声明符
在做选择时,始终使用[]和()优先于*
//
int *(*x[10])(void);
